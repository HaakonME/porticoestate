Konzept der Datenobjekte (skizziert)
------------------------------------

Speicherung der Daten in einem Buffer der Form

        array("attrname1" => array("state" => (isUnknown, isRead, isDirty),
                                  "value" => ""
                            ),
              "attrname2" => array("state" => (isUnknown, isRead, isDirty),
                                  "value" => ""
                            )
        )

Z.B. Maildomain
        array("accounts" => array("state" => (stateUnknown, stateRead, stateDirty),
                                  "value" => array(Mailaccount1, Mailaccount2))
                            )
              "domain" => array("state" => (isUnknown, isRead, isDirty),
                                  "value" => "probusiness.de"
                            )
        )

        $_Buffer["accounts"]["state"]
        $_Buffer["accounts"]["value"]

getAttribute liefert den Wert des Attributes:

        getAttribute($attr) {
            //teste, ob das attribut geladen ist
            ...
            //wert zurückgeben
            return($_Buffer[$attr]["content"]);
        }

Damit kann dann zB. wie folgt weitergearbeitet werden

        $accounts = $domain->getAttribute("accounts") //array of objects of class Mailaccount
        foreach ($accounts as $account) {
            $name = $account->getAttribute("cn");
            $mails = $account->getAttribute("mails");
        }

Oder so (auf Account)
        $account->setAttribute("set", "Andreas Schiller") //setzt dann gleichzeitig den dirty-Status
        $account->save(); //speichert die modifizierten Attribute
